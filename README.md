# Tasks Completed

## 1. Iris Flower Classification

### Project Description
This project helped me understand the basic workflow of a machine learning classification problem. The dataset contained measurements of iris flowers such as sepal length, sepal width, petal length, and petal width. Using these features, the model predicts the species of the flower:
- Setosa
- Versicolor
- Virginica

For this project, I trained different classification models including:
- K-Nearest Neighbor
- Decision Tree Classifier
- Support Vector Machine

Both models performed well and achieved good accuracy in predicting flower species.

### What I Learned
- Basics of Machine Learning Classification
- Data preprocessing techniques
- Training and testing machine learning models
- Accuracy evaluation
- Understanding feature-based prediction

## 2. Unemployment Analysis in India

## 3. Car Price Prediction

### Project Description
The objective of this project was to predict the selling price of a car using machine learning techniques. The dataset contained multiple features such as:
- Car age
- Fuel type
- Transmission type
- Present price
- Kilometers driven

Since the dataset contained categorical data, I used encoding techniques to convert them into numerical values before training the model.

I used the Linear Regression algorithm for prediction and evaluated the model using error metrics.

### What I Learned
- Regression algorithms
- Label Encoding
- Feature preprocessing
- Model evaluation using Mean Squared Error
- Understanding factors affecting car prices

## 4. Email Spam Detection

### Project Description
This project focused on building a spam detection system using machine learning. The dataset contained SMS/email messages labeled as either spam or ham (non-spam).

To process the text data:
- I used Count Vectorizer to convert text messages into numerical vectors
- Then trained the model using Multinomial Naive Bayes

The trained model was able to classify messages as spam or non-spam with high accuracy.

### What I Learned
- Natural Language Processing basics
- Text preprocessing
- Count Vectorizer implementation
- Naive Bayes algorithm
- Spam message classification

## 5. Sales Prediction Using Python

### Project Description
In this project, I analyzed advertising data to predict future sales based on advertisement spending on:

The project helped in understanding how advertisement spending influences product sales.

- Correlation analysis
- Predictive analytics
